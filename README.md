## Book Recommender
Goal: Build a collaborative-based filtering recommendation system. 

Book recommendation system that will identify similiar user behaviors and make recommendations based on what similiar users have chosen.

<b>Using:</br></b>
Django</br>
Postgresql</br>
AWS S3</br>
